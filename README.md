# Lab Gestão de Configuração

## Descrição

Este projeto foi desenvolvido como parte do laboratório da disciplina de Gestão de Configuração. O objetivo é praticar o uso do Git e do GitHub, aplicando conceitos como controle de versão, commits, branches, merges e colaboração em projetos de software.

## Funcionalidades

* Exibição da mensagem "Hello World".
* Exibição da mensagem "Adeus!".
* Controle de versão utilizando Git.
* Histórico de alterações através de commits padronizados.

## Tecnologias Utilizadas

* Git
* GitHub
* Linguagem de programação utilizada no projeto

## Objetivo Acadêmico

Este repositório foi criado para demonstrar a aplicação prática dos conceitos de Gestão de Configuração, incluindo versionamento de código, criação de branches, realização de merges e utilização de boas práticas de commits.
